# Do Julgamento à Ressurreição

Um mini guia sobre os eventos que marcaram a morte e a ressurreição de Jesus Cristo.

## Conteúdo
Este repositório apresenta um estudo simples e direto sobre os momentos mais importantes da jornada de Jesus Cristo:
- Julgamento
- Crucificação
- Sepultamento
- Ressurreição

## Objetivo
Ajudar no entendimento dos acontecimentos centrais da fé cristã, servindo como:

Guia de estudo
Material de reflexão
Base para anotações

Como usar
Leia os capítulos na ordem
Faça suas próprias anotações
Use como apoio para estudos ou devocionais
